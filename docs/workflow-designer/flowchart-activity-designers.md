# <a name="flowchart-activity-designers"></a>ActivityDesigner Diagramma di flusso
[!INCLUDE[wfd1](../workflow-designer/includes/wfd1_md.md)] include alcune attività fornite dal sistema che è possibile usare per la costruzione di flussi di lavoro. Contenuto della sezione sono presentate le attività fornite dal sistema usate per gestire diagrammi di flusso all'interno di un flusso di lavoro. Negli argomenti riportati di seguito vengono illustrate queste attività e viene fornito materiale sussidiario sulle relative modalità di utilizzo.  
  
## <a name="in-this-section"></a>Contenuto della sezione  
 [Diagramma di flusso](../workflow-designer/flowchart-activity-designer.md)  
 **Consente di creare flussi di lavoro che definiscono e gestiscono controlli di flusso complessi.**  [FlowDecision](../workflow-designer/flowdecision-activity-designer.md)  
 Fornisce due rami alternativi per il flusso di controllo a seconda che venga soddisfatta una condizione specificata.  
  
 [FlowSwitch\<T >](../workflow-designer/flowswitch-t-activity-designer.md)  
 Consente la creazione di rami per il flusso di controllo in base ai criteri di corrispondenza quando sono necessari più di due rami alternativi.
