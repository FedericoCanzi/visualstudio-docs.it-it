# Visualizzazioni dei dati su conflitti tra risorse
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

In questa sezione sono incluse informazioni di riferimento relative alle visualizzazioni e ai rapporti dei file di dati del profiler che contengono dati di profilo sui conflitti dei thread.  
  
## Argomenti della sezione  
 [Summary View \- Profiler Contention Data](../profiling/resource-contention-data-views.md)  
 Visualizza una sequenza temporale grafica dei dati di profilo ed elenca le funzioni e le risorse che hanno partecipato alla maggior parte degli eventi di blocco.  
  
 [Visualizzazione Struttura ad albero delle chiamate](../profiling/call-tree-view-contention-data.md)  
 Visualizza una struttura ad albero gerarchica che rappresenta i percorsi di esecuzione e i dati sui conflitti di risorse delle funzioni nell'esecuzione del profilo.  
  
 [Visualizzazione Moduli](../profiling/modules-view-contention-data.md)  
 Organizza i dati sui conflitti di risorse e thread in base al modulo ed elenca le funzioni, le righe del codice sorgente e le istruzioni che erano in esecuzione quando si sono verificati gli eventi di blocco.  
  
 [Visualizzazione Chiamante\/chiamato](../profiling/caller-callee-view-contention-data.md)  
 Elenca i dati sui conflitti di risorse e thread per una funzione selezionata e le funzioni chiamate dalla funzione selezionata.  
  
 [Visualizzazione Dettagli risorsa](../profiling/resource-details-view-contention-data.md)  
 Visualizza una sequenza temporale grafica degli eventi di blocco per ciascuna risorsa in conflitto ed elenca lo stack di chiamate per gli eventi di blocco.  
  
 [Visualizzazione Dettagli thread](../profiling/thread-details-view-contention-data.md)  
 Visualizza una sequenza temporale grafica degli eventi di blocco per ciascun thread ed elenca lo stack di chiamate per gli eventi di blocco.  
  
 [Visualizzazione Funzioni](../profiling/functions-view-contention-data.md)  
 Elenca i dati sui conflitti di risorse e thread in base alla funzione.  
  
 [Visualizzazione relativa ai conflitti di risorse](../profiling/resource-contentions-view-contention-data.md)  
 Elenca i dati sui conflitti di risorse in base alla risorsa bloccata.  
  
 [Visualizzazione Righe](../profiling/lines-view-contention-data.md)  
 Elenca i dati sui conflitti di risorse in base alla riga del codice sorgente.  
  
 [Visualizzazione Puntatori all'istruzione](../profiling/instruction-pointers-ips-view-contention-data.md)  
 Elenca i conflitti di risorse in base all'istruzione.  
  
 [Visualizzazione Processo](../profiling/process-view-contention-data.md)  
 Elenca i conflitti di risorse in base al processo e al thread.  
  
## Riferimento  
 [Visualizzazione Dettagli funzione](../profiling/function-details-view.md)  
 Visualizza un grafico della relazione tra una funzione selezionata e le funzioni che hanno chiamato e che sono state chiamate dalla funzione selezionata.
