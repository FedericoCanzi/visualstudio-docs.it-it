# Riferimenti per le API del profiler di Visual Studio (native)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Le API del profiler di Visual Studio consentono di controllare a livello di codice l'entità dei dati raccolti e di inserire i contrassegni di timestamp e analisi durante l'analisi.  Per utilizzare le API native, è necessario includere il file di intestazione VSPerf.h e aggiungere il file VSPerf.lib nel progetto.  
  
> [!NOTE]
>  Per impostazione predefinita, VSPerf.h e VSPerf.lib sono situati nella directory \<unità\>:\\Programmi\\Microsoft Visual Studio 9\\Team Tools\\Performance Tools\\PerfSDK.
