# Impossibile recuperare le informazioni sullo schema per l&#39;oggetto di database &lt;nome oggetto&gt;
Questo messaggio viene visualizzato in genere quando un oggetto in **Esplora server\/Esplora database** viene copiato negli Appunti, eliminato dal database e incollato nella finestra di progettazione.La visualizzazione del messaggio è pertanto dovuta al fatto che l'oggetto di database non esiste più.  
  
### Per correggere l'errore  
  
-   Scegliere **OK** per chiudere la finestra di dialogo.
