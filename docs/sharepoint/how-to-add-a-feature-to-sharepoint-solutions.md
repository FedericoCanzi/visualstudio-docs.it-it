# Procedura: aggiungere una funzionalit&#224; alle soluzioni SharePoint
  La creazione di una soluzione SharePoint prevede l'aggiunta di una funzionalità e un pacchetto.  È possibile aggiungere più elementi di funzionalità per organizzare tipi diversi di file di SharePoint.  
  
### Per aggiungere una funzionalità tramite Esplora soluzioni  
  
1.  In **Esplora soluzioni** aprire il menu di scelta rapida per **Funzionalità**, quindi scegliere **Aggiungi funzionalità**.
