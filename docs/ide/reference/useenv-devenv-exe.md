# /UseEnv (devenv.exe)
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

Viene avviato [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] e vengono caricate le variabili di ambiente nella finestra di dialogo **Directory VC\+\+**.  
  
## Sintassi  
  
```  
Devenv /useenv  
```  
  
## Esempio  
 Nell'esempio riportato di seguito viene avviato [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] e vengono caricate le variabili di ambiente nella finestra di dialogo **Directory VC\+\+**.  
  
```  
Devenv.exe /useenv  
```
