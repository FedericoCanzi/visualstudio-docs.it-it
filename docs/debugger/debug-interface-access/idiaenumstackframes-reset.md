# IDiaEnumStackFrames::Reset
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

Riporta all'inizio la sequenza di enumerazione.  
  
## Sintassi  
  
```cpp#  
HRESULT Reset();  
```  
  
## Valore restituito  
 Restituisca il valore `S_OK`.
