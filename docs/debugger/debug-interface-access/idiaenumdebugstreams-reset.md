# IDiaEnumDebugStreams::Reset
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

Reimposta una sequenza di enumerazione all'inizio.  
  
## Sintassi  
  
```cpp#  
HRESULT Reset ();  
```  
  
## Valore restituito  
 Restituisca il valore `S_OK`.
