# IDiaEnumDebugStreamData::Reset
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

Viene reimpostato all'inizio di una sequenza enumerata.  
  
## Sintassi  
  
```cpp#  
HRESULT Reset ();  
```  
  
## Valore restituito  
 Restituisca il valore `S_OK`.
