# <a name="window-menu-commands"></a>comandi del menu Finestra
Il **finestra** menu sono disponibili comandi per gestire le finestre di Spy + +.  
  
 **CASCADE**  
 Riorganizza tutte le finestre sovrapposte.  
  
 **Affianca orizzontalmente**  
 Ridisporre tutte le finestre in un modello di riquadro orizzontale.  
  
 **Affianca verticalmente**  
 Riorganizza tutte le finestre affiancate verticalmente.  
  
 **Disponi icone**  
 Riorganizza le icone di visualizzazione, se sono stati ridotti al minimo di windows.  
  
 **Chiudi / Chiudi tutto**  
 Chiude una o tutte le finestre di Spy + +.  
  
 **Aggiorna**  
 Aggiorna la visualizzazione attualmente selezionata.  
  
 **Finestra**  
 Consente la selezione di una finestra specifica.
