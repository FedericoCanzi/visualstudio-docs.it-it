# How to: Start and Stop the Message Log Display
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Quando una finestra [Visualizzazione messaggi](../debugger/messages-view.md) Ã¨ attiva, viene mostrata una scelta tra **Avvia registrazione** o **Arresta registrazione** nel menu **Messaggi** di Spy\+\+ e il corrispondente pulsante della barra degli strumenti **Avvia registrazione** o **Arresta registrazione** diventa attivo.  
  
### Per avviare la visualizzazione del log dei messaggi  
  
1.  Scegliere **Avvia registrazione** dal menu **Messaggi**.  
  
### Per arrestare la visualizzazione del log dei messaggi  
  
1.  Scegliere **Arresta registrazione** dal menu **Messaggi**.
