# <a name="run-uwp-apps-from-visual-studio"></a>Eseguire App UWP da Visual Studio
Le app UWP vengono eseguite su una vasta gamma di dispositivi e Visual Studio offre vari modi per eseguirle e interagire con tali app. L'esperienza "F5" è stata migliorata per semplificare il debug, la profilatura e il test delle app durante lo sviluppo.
