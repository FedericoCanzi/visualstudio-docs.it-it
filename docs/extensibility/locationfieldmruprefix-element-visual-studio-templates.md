Specifica i percorsi utilizzati di recente \(MRU, most recently used\) nella finestra di dialogo **Nuovo progetto** e **Aggiungi nuovo elemento**.  
  
## Sintassi  
  
```  
<LocationFieldMRUPrefix> ... </LocationFieldMRUPrefix>  
```
