# <a name="objects"></a>Oggetti
Le informazioni di riferimento seguenti vengono illustrati gli oggetti utilizzati per creare un'istanza dell'editor di testo di base.  
  
## <a name="in-this-section"></a>Contenuto della sezione  
 [Oggetto VSCodeWindow](../extensibility/vscodewindow-object.md)  
 Viene descritto l'oggetto finestra di codice, una finestra del documento specializzato che è possibile includere uno o più visualizzazioni di testo.  
  
 [Oggetto VSCodeWindowManager](../extensibility/vscodewindowmanager-object.md)  
 Descrive l'oggetto del manager di finestra di codice che viene implementato dal servizio di linguaggio ed è responsabile per la gestione di aree di controllo.  
  
 [Oggetto VSTextBuffer](../extensibility/vstextbuffer-object.md)  
 Viene descritto l'oggetto buffer di testo, che rappresenta un flusso di testo Unicode, i dati per un file.  
  
 [Oggetto VSTextView](../extensibility/vstextview-object.md)  
 Viene descritta la visualizzazione di testo, una finestra che consente agli utenti di visualizzare e modificare testo Unicode del buffer del testo.
