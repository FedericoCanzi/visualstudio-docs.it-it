# <a name="outlining-and-hidden-text-in-a-legacy-language-service"></a>Testo della struttura e nascosto in un servizio di linguaggio Legacy
Struttura consente di comprimere una sequenza di righe di testo in una singola riga. In C++, ad esempio, tutti i metodi possono essere compressi in una singola riga, che mostra solo la firma del metodo. Testo nascosto è la sequenza di righe di testo che può essere visualizzato o nascosto.
