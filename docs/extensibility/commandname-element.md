# Elemento CommandName
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Il `CommandName` elemento specifica il testo visualizzato nella categoria tastiera il **Opzioni** nella finestra di dialogo e il **comandi** nell'elenco il **Personalizza** la finestra di dialogo.  
  
## Sintassi  
  
```  
<CommandName>MyCommand</CommandName>  
```  
  
## Attributi ed elementi  
 Nelle sezioni seguenti vengono descritti gli attributi, gli elementi figlio e gli elementi padre.  
  
### Attributi  
 Nessuno.  
  
### Elementi figlio  
 Nessuno.  
  
### Elementi padre  
  
|Elemento|Descrizione|  
|--------------|-----------------|  
|[Elemento di stringhe](../extensibility/strings-element.md)|Raggruppa gli elementi di testo, ad esempio `ButtonText` e `CommandName`.|
