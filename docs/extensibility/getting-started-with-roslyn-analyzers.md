# <a name="getting-started-with-roslyn-analyzers"></a>Introduzione agli analizzatori di Roslyn
Con gli analizzatori di codice live, basato sul progetto in Visual Studio, gli autori di API possono spedire l'analisi del codice specifico di dominio come parte di ai pacchetti NuGet.  Poiché gli analizzatori sono basate sulla piattaforma del compilatore .NET (nome in codice "Roslyn"), mentre si digita prima volta la riga (non è più in attesa per compilare il codice per individuare i problemi) possono produrre avvisi nel codice.  Gli analizzatori è possibile che anche una correzione automatica di codice tramite la richiesta di lampadina di Visual Studio per consentire immediatamente il codice di pulizia
