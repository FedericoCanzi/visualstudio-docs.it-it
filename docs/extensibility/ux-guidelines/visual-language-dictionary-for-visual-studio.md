# <a name="visual-language-dictionary-for-visual-studio"></a>Dizionario di linguaggio Visual per Visual Studio
Visual Studio utilizza un linguaggio visivo comune per illustrare i concetti che sono i blocchi predefiniti di relativo visualizzato e altre immagini. Quando si progetta nuove icone, utilizzare questo dizionario per identificare i concetti che si desidera trasmettere in un'immagine personalizzata.  
  
 **Linee guida generali**  
  
-   Questi concetti visual devono essere utilizzati in modo coerente con le relative definizioni di dizionario. In caso contrario, si rischia di confondere gli utenti, che prevedono elementi comuni per hanno un significato coerenza.  
  
-   Scegliere solo i concetti più importanti o pertinenti per l'utilizzo di icone, in modo che l'immagine è conciso e facilmente distinguibili.  
  
-   Utilizzare solo uno o due concetti in ciascuna immagine.  
  
-   Per altre strategie per le icone di progettazione, vedere [progettazione icona](../../extensibility/ux-guidelines/images-and-icons-for-visual-studio.md#BKMK_IconDesign).
