# <a name="command-line-switches-visual-studio-sdk"></a>Opzioni della riga di comando (Visual Studio SDK)
Devenv.exe, il file eseguibile che avvia il [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] ambiente di sviluppo integrato (IDE), supporta i parametri della riga di comando in modo specifico per lo sviluppo di package VS. Package VS può inoltre rendere disponibili i propri parametri nella riga di comando Devenv. Gli argomenti seguenti forniscono informazioni su come utilizzare le opzioni della riga di comando nell'IDE.  
  
## <a name="in-this-section"></a>Contenuto della sezione  
 [Opzioni della riga di comando devenv per lo sviluppo di VSPackage](../extensibility/devenv-command-line-switches-for-vspackage-development.md)  
 Vengono descritte le opzioni della riga di comando Devenv che consente di semplificare VSPackage sviluppo, debug e distribuzione.  
  
 [Aggiunta della riga di comando](../extensibility/adding-command-line-switches.md)  
 Viene descritto come implementare, cercare e leggere immesso dall'utente della riga di comando per il pacchetto.  
  
## <a name="related-sections"></a>Sezioni correlate  
 [Opzioni della riga di comando devenv](../ide/reference/devenv-command-line-switches.md)  
 Viene illustrato come Devenv consente di impostare diverse opzioni per l'IDE e viene illustrato come compilare, compilare ed eseguire il debug di progetti dalla riga di comando. È disponibile anche informazioni sulla sintassi delle opzioni Devenv e una tabella di opzioni.
