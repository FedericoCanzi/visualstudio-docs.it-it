# Errore MSB1039 di MSBuild
**MSB1039: specificare la versione del set di strumenti.**  
  
 Non è stato specificato alcun valore quando è stata utilizzata l'opzione **\/ToolsVersion**.  
  
### Per correggere l'errore  
  
-   Quando si utilizza l'opzione **\/ToolsVersion**, verificare che si specifichi un valore valido, non vuoto per `ToolsVersion`.
