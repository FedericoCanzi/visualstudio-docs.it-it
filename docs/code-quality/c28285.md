# C28285
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avviso C28285: per la funzione, errore di sintassi nel parametro  
  
 Lo strumento di analisi del codice riporta questo avviso quando un possibile errore viene individuato nel file modello.  Alcuni errori del file di origine possono provocare tali errori.
