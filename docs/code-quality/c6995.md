# C6995
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Impossibile salvare il file di log XML.  
  
 L'avviso indica che lo strumento di analisi del codice non può creare il log degli errori, che rappresenta l'output dell'analisi del codice.  
  
 Questo errore potrebbe indicare un errore del disco o indicare che non si è autorizzati a creare un file nella directory specificata.
