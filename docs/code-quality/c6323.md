# C6323
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avviso C6323: utilizzo di un operatore aritmetico su tipi boolean  
  
 Questo avviso si verifica se su tipi di dati boolean vengono utilizzati operatori aritmetici.  L'utilizzo di operatori non corretti pu√≤ determinare risultati non validi  e indica inoltre che il codice non riflette lo scopo del programmatore.  
  
## Esempio  
 Il codice seguente genera questo avviso:  
  
```  
int test(bool a, bool b)  
{  
    int c = a + b;     //C6323  
    return c;  
}  
```  
  
 Per risolvere il problema, utilizzare l'operatore e il tipo di dati corretti.  
  
```  
int test(int a, int b)  
{  
    int c = a + b;       
    return c;  
}  
```
