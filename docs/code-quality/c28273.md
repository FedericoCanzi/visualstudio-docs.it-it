# C28273
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

warning C28273: Per la funzione, le indicazioni non sono coerenti con la dichiarazione della funzione  
  
 L'avviso indica un errore nelle annotazioni, non nel codice che sta analizzando.  Le annotazioni visualizzate in una definizione di funzione sono contraddittorie con quelle visualizzate in una dichiarazione.  Le due annotazioni devono essere risolte per ottenere la corrispondenza.
