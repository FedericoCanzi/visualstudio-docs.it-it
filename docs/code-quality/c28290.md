# C28290
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

warning C28290: l'annotazione per la funzione contiene più External del numero di parametri effettivi  
  
 Lo strumento di analisi del codice riferisce questo avviso quando l'annotazione per la funzione contiene più Externals del numero attuale di parametri.
