# C28224
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

warning C28224: l'annotazione richiede dei parametri  
  
 L'avviso indica che l'annotazione denominata viene utilizzata senza parametri e almeno un parametro è obbligatorio.  Ciò non dovrebbe essere possibile se sono state utilizzate le macro standard per le annotazioni; l'avviso non dovrebbe venir visualizzato nell'utilizzo normale.
