# C28228
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

warning C28228: Parametro di annotazione: impossibile utilizzare tipi nelle annotazioni  
  
 Questo avviso indica che un parametro è di un tipo non supportato.  Le annotazioni possono utilizzare solo un set limitato di tipi come parametri.  Ciò non si verifica se le macro standard vengono utilizzate per le annotazioni; questo avviso non è previsto che venga visualizzato nell'utilizzo tipico.
