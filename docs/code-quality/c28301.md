# C28301
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avviso C28301: nessuna annotazione per la prima dichiarazione di \<funzione\>.\<nota1\> Vedere \<nome file\>\(\<riga\>\). \<nota2\>  
  
 Questo avviso viene visualizzato quando le annotazioni non vengono trovate nella prima dichiarazione di una data funzione.
