# C28231
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

warning C28231: l'annotazione è valida solo su array  
  
 Questo avviso indica che un argomento ad un'annotazione deve essere un array e qualche altro tipo è stato rilevato.
