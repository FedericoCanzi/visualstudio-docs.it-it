# C28225
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avviso C28225: Non ha trovato il numero corretto di parametri necessari nell'annotazione  
  
 L'avviso indica che la voce denominata viene utilizzata con un numero di parametri errato.  Ciò non si verifica se le macro standard vengono utilizzate per le annotazioni; questo avviso non è previsto che venga visualizzato nell'utilizzo tipico.
