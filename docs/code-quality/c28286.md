# C28286
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

warning C28286: Per la funzione, errore di sintassi vicino alla fine  
  
 Lo strumento di analisi del codice riferisce questo avviso quando un possibile errore viene individuato nel file modello.  Alcuni errori del file di origine possono provocare tali errori.
