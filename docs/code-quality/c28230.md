# C28230
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avviso C28230: il tipo di parametro non ha alcun membro.  
  
 L'avviso indica che un argomento a un'annotazione tenta di accedere a `struct`, `class`, o `union` e il membro denominato non esiste.
