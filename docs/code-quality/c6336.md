# C6336
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avviso C6336: l'operatore aritmetico ha precedenza sull'operatore punto interrogativo. Utilizzare delle parentesi per chiarire lo scopo  
  
 L'avviso indica un possibile problema di precedenza tra operatori.  Gli operatori '\+','\-','\*' e '\/' hanno la precedenza sull'operatore '?'.  Se la precedenza dell'operatore nell'espressione non Ã¨ corretta, utilizzare le parentesi per modificarla.  
  
## Esempio  
 Il codice seguente genera questo avviso:  
  
```  
int Count();  
  
void f(int flag)  
{  
  int result;  
  result = Count() + flag ? 1 : 2;  
  // code...  
}  
```  
  
 Per risolvere il problema, aggiungere le parentesi come riportato nel codice seguente:  
  
```  
int Count();  
  
void f(int flag)  
{  
  int result;  
  result = Count() + (flag ? 1 : 2);  
  // code...  
}  
```
