# C28211
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

warning C28211: Nome di contesto statico previsto per il SAL\_context  
  
 L'avviso indica che l'operando dell'annotazione `_Static_context_` deve corrispondere al nome di un contesto definito nel tool.  L'avviso non dovrebbe mai presentarsi nell'utilizzo normale.
