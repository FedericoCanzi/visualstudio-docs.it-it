# C28300
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

warning C28300: \<parameter\_name\>: Operandi dell'espressione di tipi incompatibili per l'operatore \<operator\_name\>  
  
 Questo avviso viene visualizzato quando i tipi dell'operando in un parametro non sono compatibili con l'operatore.
