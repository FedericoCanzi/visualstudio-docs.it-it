# C28223
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warning C28223: Non è stato trovato il token\/identificatore previsto per il parametro di annotazione  
  
 L'avviso indica che un parametro previsto ad un'annotazione sia un identificatore, e un altro simbolo è stato rilevato.  Ciò indica in genere una macro codificata in modo errato.  L'utilizzo delle parole chiave C o C\+\+ in questa posizione provocherà questo errore.
