# C28222
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

warning 28222: \_Yes\_, \_No\_, o \_Maybe\_ previsto per l'annotazione  
  
 Questo warning indica che un parametro ad un'annotazione deve essere uno dei simboli `_Yes_`, `_No_`, o `_Maybe_` e qualche altro simbolo è stato rilevato.  Ciò indica in genere una macro codificata in modo errato.
