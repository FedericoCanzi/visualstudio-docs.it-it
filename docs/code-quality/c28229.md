# C28229
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

warning C28229: L'annotazione non supporta parametri  
  
 Questo warning indica che un'annotazione è stata utilizzata con un parametro quando l'annotazione è dichiarata senza parametri.  Ciò non si verifica se le macro standard vengono utilizzate per le annotazioni; questo avviso non è previsto che venga visualizzato nell'utilizzo tipico.
