# C28209
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

warning C28209: La dichiarazione per il simbolo presenta una dichiarazione in conflitto  
  
 L'avviso indica una dichiarazione dell'annotazione costruita in modo errato.  L'avviso non deve mai presentarsi nell'utilizzo normale.
