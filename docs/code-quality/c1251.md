# C1251
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avviso C1251: Impossibile caricare i modelli.  
  
 Lo strumento di analisi del codice riferisce questo avviso quando si verifica un errore interno del file modello, non nel codice analizzato.
