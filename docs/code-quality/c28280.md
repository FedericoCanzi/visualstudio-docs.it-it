# C28280
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

warning C28280: Per il simbolo, è stato trovato un 'end' senza un 'begin' corrispondente.  
  
 Il linguaggio di annotazione supporta un costrutto begin e uno end \(`{` e `}` in C\), e l'accoppiamento è sbilanciato.  Questa situazione può essere evitata se si usano le macro.
