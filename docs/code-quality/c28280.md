# <a name="c28280"></a>C28280
avviso C28280: per il simbolo, un 'end' è stato trovato senza un corrispondente 'begin'  
  
 Il linguaggio di annotazione supporta un inizio e fine (`{` e `}` in C) costrutto e l'associazione è diventato sbilanciata. Questa situazione può essere evitata se vengono utilizzate le macro.
