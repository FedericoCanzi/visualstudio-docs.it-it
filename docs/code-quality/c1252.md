# C1252
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

warning C1252: dipendenza circolare o mancante tra plugin: richiede GUID  
  
 Lo strumento di analisi del codice riferisce questo avviso quando si verifica un errore interno con dipendenze dai plugin, non nel codice che si sta analizzando.
