# C1253
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

warning C1253: Impossibile caricare il file di modello.  
  
 Lo strumento di analisi del codice riferisce questo avviso quando si verifica un errore interno nel file modello, non nel codice analizzato.
