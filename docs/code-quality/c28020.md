# C28020
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

warning C28020: l'espressione \<expr\> non è vera in questa chiamata  
  
 Questo avviso viene visualizzato quando l'espressione \_Satisfies\_ elencata non è vera.  Spesso questo indica un parametro non corretto.  
  
 Se si verifica su una dichiarazione di funzione, le annotazioni indicano una condizione impossibile.
