# C28221
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avviso C28221: prevista espressione stringa per il parametro nell'annotazione  
  
 L'avviso indica che un parametro a un'annotazione è atteso come stringa e un altro tipo è stato rilevato.  Ciò indica in genere una macro di annotazione codificata in modo errato e non si prevede di vederla nell'utilizzo normale.
