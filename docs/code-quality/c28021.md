# C28021
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

warning C28021: il parametro \<param\> annotato con \<anno\> deve essere un puntatore  
  
 Questo avviso viene visualizzato quando l'oggetto che viene annotato non è di tipo puntatore.  Questa voce non può essere utilizzata con `void` o con i tipi integrali.
