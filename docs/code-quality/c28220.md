# C28220
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avviso C28220: prevista espressione Integer per annotare il parametro nell'annotazione  
  
 L'avviso indica che un parametro a un'annotazione è previsto che sia un'espressione Integer, e qualche altro tipo è stato rilevato.  Ciò indica in genere una macro codificata in modo errato.
