# C28160
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

warning C28160: Errore di annotazione  
  
 Questo avviso viene visualizzato quando un'annotazione `__drv_error` è stata rilevata.  Questa annotazione è usata per segnalare le tecniche di codifica che devono essere risolte e può essere utilizzata con un'annotazione `__drv_when` per indicare delle specifiche combinazioni di parametri.
