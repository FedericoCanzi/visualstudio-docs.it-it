# C28219
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

warning C28219: Membro di enumerazione previsto per annotare il parametro nell'annotazione  
  
 Un parametro ad un'annotazione è un membro del tipo denominato `enum` e un altro simbolo è stato rilevato; utilizzare un membro del tipo `enum`.  Ciò indica in genere una macro codificata in modo errato.
