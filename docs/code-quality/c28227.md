# C28227
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Avviso C28227: L'annotazione non può essere anche un PrimOp \(vedere la dichiarazione precedente\).  
  
 L'avviso indica che la voce è l'annotazione denominata è stata dichiarata annotazione comune e che era stata precedentemente dichiarata come PrimOp.  Ciò non si verifica se le macro standard vengono utilizzate per le annotazioni; non si prevede che questo avviso venga visualizzato nell'utilizzo tipico.
