# <a name="ca0055"></a>CA0055
Impossibile caricare il file:\<percorso >  
  
 Lo strumento di analisi codice non è riuscito a caricare il file di destinazione che è stato specificato per l'analisi. Questo errore si verifica se il file non è stato trovato o è stato negato l'accesso al file. In genere, questo errore si verifica se l'utente sceglie di eseguire l'override della proprietà dell'assembly di input in [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)], o richiamato fxcopcmd.exe manualmente.
