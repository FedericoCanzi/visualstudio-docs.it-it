# C28275
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avviso C28275: Il parametro per \_Macro\_value\_ è null  
  
 L'avviso indica che esiste un errore interno nel file modello, non nel codice analizzato.  La funzione *macroValue* è stata richiamata senza un parametro.
