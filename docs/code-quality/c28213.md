# C28213
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avviso C28213: l'annotazione `_Use_decl_annotations_` deve essere utilizzata per fare riferimento, senza modifiche, a una dichiarazione precedente. \<why\>  
  
 Questo avviso viene visualizzato quando viene fatto riferimento ad una dichiarazione precedente senza l'annotazione richiesta `_Use_decl_annotations_` .
