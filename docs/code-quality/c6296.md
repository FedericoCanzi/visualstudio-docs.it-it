# C6296
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avviso C6296: Ciclo For definito in modo non corretto: corpo del ciclo eseguito solo una volta  
  
 L'avviso indica che un ciclo For potrebbe non funzionare come desiderato.  Quando l'indice Ã¨ senza segno e viene eseguito il conteggio alla rovescia di un ciclo a partire da zero, il corpo viene eseguito solo una volta.  
  
## Esempio  
 Il codice seguente genera questo avviso:  
  
```  
void f( )  
{  
   unsigned int i;  
  
   for (i = 0; i < 100; i--)  
   {  
      // code ...  
   }  
}  
```  
  
 Per risolvere il problema, utilizzare il codice seguente:  
  
```  
void f( )  
{  
   unsigned int i;  
  
   for (i = 0; i < 100; i++)  
   {  
      // code ...  
   }  
}  
```
