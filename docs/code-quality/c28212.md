# C28212
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

avviso C28212: richiesto un puntatore ad espressione per l'annotazione  
  
 Questo avviso indica che è previsto che il parametro numerato a un'annotazione \(non la funzione che viene annotata\) sia un puntatore o un array, ma è stato rilevato un altro tipo.  L'annotazione deve essere corretta.
