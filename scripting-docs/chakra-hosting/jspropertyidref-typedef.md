# Typedef JsPropertyIdRef
Identificatore di proprietà.  
  
## Sintassi  
  
```  
typedef JsRef JsPropertyIdRef;  
```  
  
## Note  
 Gli identificatori di proprietà vengono usati per fare riferimento alle proprietà degli oggetti JavaScript anziché usare le stringhe.  
  
## Requisiti  
 **Intestazione:** jsrt.h
