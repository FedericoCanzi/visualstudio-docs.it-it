# Funzione Math.sqrt (JavaScript)
Restituisce la radice quadrata di un numero.  
  
## Sintassi  
  
```  
  
Math.sqrt(  
number  
)   
```  
  
## Note  
 L'argomento `number` obbligatorio è un'espressione numerica.  
  
 Se `number` è negativo, il valore restituito sarà `NaN`.
