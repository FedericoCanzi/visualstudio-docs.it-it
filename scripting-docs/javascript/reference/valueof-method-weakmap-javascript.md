# Metodo valueOf (WeakMap) (JavaScript)
Restituisce il valore primitivo dell'oggetto specificato.  
  
## Sintassi  
  
```javascript  
weakmapObj.valueOf()  
```  
  
#### Parametri  
 `weakmapObj`  
 Necessario.  Un oggetto `WeakMap`.

 Restituisce l'istanza dell'oggetto `WeakMap`.
