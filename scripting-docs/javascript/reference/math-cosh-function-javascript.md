# Funzione Math.cosh (JavaScript)
Restituisce il coseno iperbolico di un numero.  
  
## Sintassi  
  
```  
  
Math.cosh(x)   
```  
  
## Note  
 Il valore restituito è lo stesso di `(exp(x) + exp(-x))/2`.  Se `x` è `NaN`, il risultato è `NaN`.  
  
 **Si applica a**: [Oggetto Math](../../javascript/reference/math-object-javascript.md)
