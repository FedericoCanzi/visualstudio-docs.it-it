# Operatore di divisione (/) (JavaScript)
Consente di dividere i valori di due espressioni.  
  
## Sintassi  
  
```  
  
result = number1 / number2   
```  
  
## Parametri  
 `result`  
 Qualsiasi variabile numerica.  
  
 `number1`  
 Qualsiasi espressione numerica.  
  
 `number2`  
 Qualsiasi espressione numerica.
