# Funzione Math.tanh (JavaScript)
Restituisce la tangente iperbolica di un numero.  
  
## Sintassi  
  
```  
  
Math.tanh(  
x  
)  
  
```  
  
## Note  
 Il valore restituito è lo stesso di `(exp(x) - exp(-x))/(exp(x) + exp(-x))`.  Se `x` è `NaN`, il risultato è `NaN`.
