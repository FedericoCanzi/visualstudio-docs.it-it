# Funzione Math.atan2 (JavaScript)
Restituisce l'angolo, espresso in radianti, compreso tra l'asse X e un punto \(*y*,*x*\).  
  
## Sintassi  
  
```  
  
Math.atan2(y, x)  
```  
  
## Parametri  
 *x*  
 Obbligatorio.  Espressione numerica che rappresenta la coordinata cartesiana x.  
  
 *y*  
 Obbligatorio.  Espressione numerica che rappresenta la coordinata cartesiana y.
