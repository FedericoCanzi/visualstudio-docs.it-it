# Funzione Math.log2 (JavaScript)
Restituisce il logaritmo in base 2 di un numero.  
  
## Sintassi  
  
```  
Math.log2(x)   
```  
  
## Valore restituito  
 Se `x` è `NaN`, questa funzione restituisce `NaN`.  Se `x` è minore di zero \(0\), questa funzione restituisce `NaN`.
