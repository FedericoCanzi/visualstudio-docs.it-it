# Funzione Math.fround (JavaScript)
Restituisce il formato in virgola mobile a precisione singola più vicino di un numero.  
  
## Sintassi  
  
```  
  
Math.fround(  
number  
)   
```  
  
## Note  
 Se `number` è `NaN`, il risultato è `NaN`.
