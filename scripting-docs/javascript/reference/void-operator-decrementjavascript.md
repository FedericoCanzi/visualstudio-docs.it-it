# Operatore void (JavaScript)
Impedisce la restituzione di un valore da parte di un'espressione.  
  
## Sintassi  
  
```  
void expression   
```  
  
## Note  
 L'argomento `expression` rappresenta qualsiasi espressione [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] valida.  
  
 L'operatore `void` valuta l'espressione relativa e restituisce `undefined`.  Ãˆ utile per valutare un'espressione senza che il risultato sia visibile nel resto dello script.
