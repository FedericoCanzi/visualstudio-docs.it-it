# Propriet&#224; size (Map) (JavaScript)
Restituisce il numero di elementi in una mappa.  
  
## Sintassi  
  
```javascript  
sizeVar = mapObj.size  
```  
  
#### Parametri  
 `sizeVar`  
 Necessario.  Qualsiasi numero.  
  
 `mapObj`  
 Necessario.  Qualsiasi oggetto `Map`.
