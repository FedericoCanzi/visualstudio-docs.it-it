# Metodo valueOf (Set) (JavaScript)
Restituisce il valore primitivo dell'oggetto specificato.  
  
## Sintassi  
  
```javascript  
setObj.valueOf()  
```  
  
#### Parametri  
 `setObj`  
 Necessario.  Un oggetto `Set`.

 Restituisce l'istanza del set.
