# Operatore di assegnazione di modulo (JavaScript)
Consente di dividere il valore di una variabile per il valore di un'espressione e di assegnare il resto della divisione alla variabile.  
  
## Sintassi  
  
```  
  
result %= expression  
```  
  
## Argomenti  
 `result`  
 Qualsiasi variabile.  
  
 `expression`  
 Qualsiasi espressione numerica.  
  
## Note  
 L'utilizzo dell'operatore `%=` equivale esattamente a specificare:  
  
```  
result = result % expression  
```
