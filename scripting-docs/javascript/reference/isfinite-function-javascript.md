# Funzione isFinite (JavaScript)
Determina se il numero specificato è finito.  
  
## Sintassi  
  
```  
isFinite(number)   
```  
  
## Note  
 L'argomento `number` richiesto è un valore numerico.  
  
 La funzione `isFinite` restituisce `true` se `number` è un qualsiasi valore diverso da `NaN`, un infinito negativo o un infinito positivo.  In caso contrario, viene restituito **false**.
