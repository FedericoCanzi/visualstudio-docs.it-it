# Metodo valueOf (Map) (JavaScript)
Restituisce il valore primitivo dell'oggetto specificato.  
  
## Sintassi  
  
```javascript  
mapObj.valueOf()  
```  
  
#### Parametri  
 `mapObj`  
 Necessario.  Un oggetto `Map`.
