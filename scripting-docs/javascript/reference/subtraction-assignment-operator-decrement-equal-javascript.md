# Operatore di assegnazione di sottrazione (-=) (JavaScript)
Consente di sottrarre il valore di un'espressione dal valore di una variabile e di assegnare il risultato alla variabile.  
  
## Sintassi  
  
```  
  
result -= expression  
```  
  
## Parametri  
 *result*  
 Qualsiasi variabile numerica.  
  
 *expression*  
 Qualsiasi espressione numerica.  
  
## Note  
 Utilizzare l'operatore `-=` equivale esattamente a eseguire la seguente operazione:  
  
```javascript  
result = result â€“ expression  
```
