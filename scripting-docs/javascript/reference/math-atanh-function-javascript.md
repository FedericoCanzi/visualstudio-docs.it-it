# Funzione Math.atanh (JavaScript)
Restituisce la tangente iperbolica inversa di un numero.  
  
## Sintassi  
  
```  
  
Math.atanh(x)   
```  
  
## Note  
 Se `x` è `NaN`, il risultato è `NaN`.  
  
 **Si applica a**: [Oggetto Math](../../javascript/reference/math-object-javascript.md)
