# Funzione Math.log10 (JavaScript)
Restituisce il logaritmo in base 10 di un numero.  
  
## Sintassi  
  
```  
Math.log10(x)   
```  
  
## Valore restituito  
 Se `x` è `NaN`, questa funzione restituisce `NaN`.  Se `x` è minore di zero \(0\), questa funzione restituisce `NaN`.
