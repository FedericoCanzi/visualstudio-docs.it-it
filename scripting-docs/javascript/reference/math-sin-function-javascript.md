# Funzione Math.sin (JavaScript)
Restituisce il seno di un numero.  
  
## Sintassi  
  
```  
  
Math.sin(radians)   
```  
  
## Note  
 L'argomento `radians` Ã¨ un'espressione numerica contenente un angolo espresso in radianti.  
  
 Il valore restituito corrisponde al seno dell'argomento numerico.
