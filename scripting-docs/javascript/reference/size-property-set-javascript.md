# Propriet&#224; size (Set) (JavaScript)
Restituisce il numero di elementi in un set.  
  
## Sintassi  
  
```javascript  
sizeVar = setObj.size  
```  
  
#### Parametri  
 `sizeVar`  
 Necessario.  Qualsiasi numero.  
  
 `setObj`  
 Necessario.  Qualsiasi oggetto `Set`.
