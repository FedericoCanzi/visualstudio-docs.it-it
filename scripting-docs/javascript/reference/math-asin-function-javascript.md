# Funzione Math.asin (JavaScript)
Restituisce l'arcoseno di un numero.  
  
## Sintassi  
  
```  
  
Math.asin(number)   
```  
  
## Note  
 L'argomento `number` obbligatorio è un'espressione numerica per la quale è necessario l'arcoseno.  
  
 Il valore restituito corrisponde all'arcoseno dell'argomento `number` espresso in radianti.  
  
 **Si applica a**: [Oggetto Math](../../javascript/reference/math-object-javascript.md)
