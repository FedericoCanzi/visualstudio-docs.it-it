# Funzione Math.cbrt (JavaScript)
Restituisce la radice cubica di un numero.  
  
## Sintassi  
  
```  
  
Math.cbrt(  
number  
)   
```  
  
## Note  
 Se `number` è `NaN`, il risultato è `NaN`.
