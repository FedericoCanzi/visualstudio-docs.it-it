# Funzione Math.log1p (JavaScript)
Restituisce il logaritmo naturale di 1 \+ un numero.  
  
## Sintassi  
  
```  
Math.log1p(x)   
```  
  
## Valore restituito  
 Se `x` è `NaN`, questa funzione restituisce `NaN`.  Se `x` è minore di zero \(0\), questa funzione restituisce `NaN`.
