# Operatore di assegnazione di moltiplicazione (*=) (JavaScript)
Moltiplica il valore di una variabile per il valore di un'espressione e assegna il risultato alla variabile.  
  
## Sintassi  
  
```  
  
result *= expression  
```  
  
## Parametri  
 *result*  
 Qualsiasi variabile.  
  
 *expression*  
 Qualsiasi espressione.  
  
## Note  
 Utilizzare l'operatore **\*\=** equivale esattamente a specificare:  
  
```javascript  
result = result * expression  
```
