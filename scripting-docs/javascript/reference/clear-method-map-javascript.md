# Metodo clear (Map) (JavaScript)
Rimuove tutti gli elementi da una mappa.  
  
## Sintassi  
  
```javascript  
mapObj.clear()  
```  
  
#### Parametri  
 `mapObj`  
 Necessario.  Mappa da cancellare.
