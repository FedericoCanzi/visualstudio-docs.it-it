# Operatore di moltiplicazione (*) (JavaScript)
Consente moltiplicare i valori di due espressioni.  
  
## Sintassi  
  
```  
  
result = number1*number2  
```  
  
## Parametri  
 *result*  
 Qualsiasi variabile.  
  
 `number1`  
 Qualsiasi espressione.  
  
 `number2`  
 Qualsiasi espressione.
