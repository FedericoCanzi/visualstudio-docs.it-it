# Metodo clear (WeakMap) (JavaScript)
Rimuove tutti gli elementi da un oggetto `WeakMap`.  
  
## Sintassi  
  
```javascript  
weakmapObj.clear()  
```  
  
#### Parametri  
 `weakmapObj`  
 Necessario.  Oggetto `WeakMap` da cancellare.
