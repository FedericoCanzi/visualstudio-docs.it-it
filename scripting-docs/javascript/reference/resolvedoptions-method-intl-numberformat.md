# Metodo resolvedOptions (Intl.NumberFormat)
Restituisce un oggetto in cui sono contenuti le proprietà e i valori dell'oggetto formattatore `NumberFormat`.  
  
## Sintassi  
  
```  
numberFormatObj.resolvedOptions()  
```  
  
#### Parametri  
 `numberFormatObj`  
 L'oggetto `NumberFormat` da cui recuperare informazioni.  
  
## Note  
 Le proprietà dell'oggetto restituito corrispondono alle proprietà calcolate dell'oggetto `NumberFormat`.
