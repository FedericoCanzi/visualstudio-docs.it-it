# Previsto &#39;=&#39; (JavaScript)
Si è tentato di creare una variabile da utilizzare con istruzioni di compilazione condizionale, ma non è stato posizionato un segno di uguale tra la variabile e il valore da assegnarle.  
  
### Per correggere l'errore  
  
-   Aggiungere un segno di uguale.  Di seguito è riportato un esempio:  
  
    ```javascript  
    @set @myvar1 = 1  
    ```
