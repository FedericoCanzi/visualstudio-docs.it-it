# Previsto &#39;)&#39; (JavaScript)
Si è tentato di racchiudere un'espressione in un set di parentesi senza includere la parentesi di chiusura.  Una determinata espressione deve essere racchiusa in un set di parentesi di apertura e di chiusura.  L'utilizzo di parentesi è illustrato nell'esempio seguente.  
  
```javascript  
for (initialize; test; increment) {  
statement;  
}  
```  
  
### Per correggere l'errore  
  
-   Aggiungere le parentesi chiuse all'espressione di valutazione.
