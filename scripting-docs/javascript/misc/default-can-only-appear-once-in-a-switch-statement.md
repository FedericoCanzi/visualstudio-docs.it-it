# &#39;default&#39; pu&#242; apparire una sola volta in un&#39;istruzione &#39;switch&#39;
Hai tentato di utilizzare più volte l'istruzione **default** all'interno di un'istruzione switch.  Il caso predefinito, ovvero il caso di passaggio, è sempre l'ultima istruzione case in un'istruzione switch.  
  
### Per correggere l'errore  
  
-   Rimuovi tutte le istruzioni case **default** aggiuntive dall'istruzione `switch`. Utilizza al massimo un'istruzione case predefinita nell'istruzione switch.
