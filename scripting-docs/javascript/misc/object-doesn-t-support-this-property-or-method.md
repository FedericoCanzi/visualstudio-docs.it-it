# L&#39;oggetto non supporta la propriet&#224; o il metodo corrente
È stato specificato un metodo o una proprietà non esistente per questo oggetto.  Non tutti gli oggetti supportano tutte le proprietà e tutti i metodi.  
  
### Per correggere l'errore  
  
-   Controllare che le proprietà e i metodi non contengano errori di digitazione.  
  
-   Verificare che il tipo di runtime dell'oggetto sia il tipo di oggetto previsto.  
  
-   Per altre informazioni, vedere la documentazione dell'oggetto.
