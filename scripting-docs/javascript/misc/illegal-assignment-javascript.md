# Assegnazione non valida (JavaScript)
Si è tentato di assegnare un valore a un identificatore in sola lettura.  Non si può assegnare un valore a un identificatore in sola lettura.  Ad esempio, oggetti definiti dall'host e oggetti COM esterni.  
  
### Per correggere l'errore  
  
-   Non assegnare valori a identificatori in sola lettura.
