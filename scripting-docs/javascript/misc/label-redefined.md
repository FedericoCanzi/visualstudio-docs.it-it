# Etichetta ridefinita
È stata creata una nuova etichetta a cui è stato assegnato il nome di un'etichetta esistente.  È possibile utilizzare le etichette per contrassegnare blocchi di codice, ma all'interno di un ambito specificato devono essere univoche.  
  
### Per correggere l'errore  
  
-   Assicurarsi che tutte le etichette utilizzate nei programmi siano univoche nei rispettivi ambiti.
