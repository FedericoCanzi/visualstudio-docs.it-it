# Impossibile trovare l&#39;etichetta
Si è fatto riferimento a un'etichetta che non esiste.  È possibile utilizzare le etichette per contrassegnare blocchi di codice, ma all'interno di un ambito specificato devono essere univoche.  
  
### Per correggere l'errore  
  
-   Controllare per accertarsi di avere digitato correttamente il nome dell'etichetta.  
  
-   Verificare che tutti i riferimenti alle etichette rimandino a quelle definite nell'ambito corrente, incluse le definizioni con prototipo.
