# IEnumRemoteDebugApplicationThreads::Reset
Riporta all'inizio la sequenza di enumerazione.  
  
## Sintassi  
  
```  
HRESULT Reset();  
```  
  
#### Parametri  
 Questo metodo non accetta parametri.
