# IDebugPropertyEnumType_All::GetName
Restituisce un BSTR contenente il nome `EnumType`.  
  
## Sintassi  
  
```  
HRESULT GetName(  
   BSTR*  pname  
);  
```  
  
#### Parametri  
 `pname`  
 \[out\] A BSTR contenente il nome `EnumType`.  
  
## Valore restituito  
 Restituisce `HRESULT`valido, in genere `S_OK`.
