# Interfaccia IBindEventHandler
Consente al chiamante di specificare un oggetto che gestisce un gestore eventi specificato.  
  
## Metodi  
 Oltre ai metodi ereditati da `IUnknown`, l'interfaccia `IBindEventHandler` espone i metodi seguenti.  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[IBindEventHandler::BindHandler](../../winscript/reference/ibindeventhandler-bindhandler.md)|Associare un evento a un oggetto.|
