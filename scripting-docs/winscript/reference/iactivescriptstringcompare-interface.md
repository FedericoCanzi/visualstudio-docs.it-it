# Interfaccia IActiveScriptStringCompare
L'interfaccia `IActiveScriptCompare` viene utilizzata per definire il metodo di confronto di stringhe per il modulo di gestione di script.  
  
## Metodi nell'ordine Vtable  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[IActiveScriptStringCompare::StrComp](../../winscript/reference/iactivescriptstringcompare-strcomp.md)|Definisce il metodo di confronto di stringhe per il modulo di gestione di script.|
