# Interfaccia IActiveScriptSiteTraceInfo
Fornisce un metodo per inviare le informazioni di traccia dello script.  
  
## Metodi  
  
|||  
|-|-|  
|Metodo|Descrizione|  
|[Metodo IActiveScriptSiteTraceInfo::SendScriptTraceInfo](../../winscript/reference/iactivescriptsitetraceinfo-sendscripttraceinfo-method.md)|Invia le informazioni di traccia che includono il tipo di evento, il contesto e l'istruzione dello script.|
