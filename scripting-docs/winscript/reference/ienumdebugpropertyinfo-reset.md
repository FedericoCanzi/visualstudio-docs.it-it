# IEnumDebugPropertyInfo::Reset
Riporta all'inizio la sequenza di enumerazione.  
  
## Sintassi  
  
```  
HRESULT Reset (  
void  
);  
```  
  
## Valore restituito  
 Restituisce `HRESULT`valido, in genere `S_OK`.
