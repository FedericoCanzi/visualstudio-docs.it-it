# IActiveScriptWinRTErrorDebug::GetCapabilitySid
Restituisce il PROCESSO di funzionalità di Windows errore di Runtime, se disponibile.  
  
> [!IMPORTANT]
>  [Interfaccia IActiveScriptWinRTErrorDebug](../../winscript/reference/iactivescriptwinrterrordebug-interface.md) implementato da PDM v11.0 e maggiore.  Trovato in activdbg100.h.  
  
## Sintassi  
  
```cpp  
HRESULT GetCapabilitySid([out] BSTR * capabilitySid);  
```  
  
#### Parametri  
 `capabilitySid`  
 Il PROCESSO di funzionalità dell'errore.
