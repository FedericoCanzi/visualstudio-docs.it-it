# Interfaccia IActiveScriptAuthorProcedure
Oltre ai metodi ereditati da `IUnknown`, l'interfaccia `IScriptAuthorProcedure` espone il metodo seguente.  
  
## Metodi nell'ordine Vtable  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[IActiveScriptAuthorProcedure::ParseProcedureText](../../winscript/reference/iactivescriptauthorprocedure-parseproceduretext.md)|Analizza un codice di creazione, aggiungere il codice di creazione al motore di creazione di script e crea un oggetto `IScriptEntry` che corrisponde al codice di creazione.|
