# Metodo IJsDebugFrame::GetName
Ottiene il nome descrittivo dell'utente dello stack frame.  
  
## Sintassi  
  
```  
HRESULT GetName(  
   BSTR *pName  
);  
```  
  
#### Parametri  
 `pName`  
 \[out\] Nome dello stack frame.
