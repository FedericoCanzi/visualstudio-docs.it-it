# Metodo IJsEnumDebugProperty::GetCount

## Sintassi  
  
```  
HRESULT GetCount(  
   ULONG *pCount  
);  
```  
  
#### Parametri  
 `pCount`  
 \[out\] Numero delle proprietà.
