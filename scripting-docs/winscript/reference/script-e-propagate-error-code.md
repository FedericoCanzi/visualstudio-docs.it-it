# Codice di errore SCRIPT_E_PROPAGATE
Un errore di script sta propagando al chiamante, che potrebbe trovarsi in un altro thread.  L'host deve passare il codice di errore al chiamante.  
  
## Sintassi  
  
```cpp  
SCRIPT_E_PROPAGATE  0x80020102L  
```
