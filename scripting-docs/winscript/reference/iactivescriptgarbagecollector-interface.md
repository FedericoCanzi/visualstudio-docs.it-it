# Interfaccia IActiveScriptGarbageCollector
Fornisce un metodo a Garbage Collection iniziale.  Questa interfaccia deve essere implementata nel motore di script attivi che desiderano eseguire la pulizia delle risorse.  
  
## Metodi  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[IActiveScriptGarbageCollector::CollectGarbage](../../winscript/reference/iactivescriptgarbagecollector-collectgarbage.md)|Avvia il processo di Garbage Collection.|
