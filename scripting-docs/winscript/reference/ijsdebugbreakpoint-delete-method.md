# Metodo IJsDebugBreakPoint::Delete
Elimina il punto di interruzione.  
  
## Sintassi  
  
```  
HRESULT Delete(void);  
```  
  
## Valore restituito  
  
## Note  
 Restituisce S\_FALSE se chiamato su un punto di interruzione eliminato.  
  
## Requisiti  
 **Intestazione:** jscript9diag.h
