# Metodo IJsDebugBreakPoint::Enable
Abilita il Punto di interruzione  
  
## Sintassi  
  
```  
HRESULT Enable(void);  
```  
  
## Valore restituito  
  
## Note  
 Restituisce E\_UNEXPECTED se chiamato su un punto di interruzione eliminato.  
  
## Requisiti  
 **Intestazione:** jscript9diag.h
